# Проект: лендинг AI-лидогенерация из Telegram

## Стек
- HTML + Tailwind CSS + vanilla JavaScript
- Одностраничный лендинг, без фреймворков
- Адаптив: mobile-first (320px → 1440px)

## Стиль
- Тёмный hero, светлые контентные блоки
- Акцент: оранжевый #FF6B35
- Шрифт: Manrope (Google Fonts)
- Референсы: linear.app, vercel.com
- Без стоковых фото, только UI-скриншоты и иконки

## Правила
- Весь код в одном файле index.html (inline styles и scripts)
- Tailwind подключать через CDN
- Иконки: Lucide Icons через CDN
- Текст на русском языке
- Никаких англицизмов без необходимости (ФЗ-168)
